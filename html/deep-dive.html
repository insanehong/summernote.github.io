API

summernote

You can initialize summernote with summernote.

$('.summernote').summernote();

External API access

Then You can use editor API through the summernote method. This is a example code for insert 'hello world' text.

$('.summernote').summernote('editor.insertText', 'hello world'));

It's is called editor module's insertText method with 'hello world'. First argument is a string type represent module and it's method. And rest are method's arguments.

API is only work for editor module

For now, API is only works for editor module. If you call API without module name, `editor.{{method}}` will be called. ex) $('.summernote').summernote('insertText', 'hello world');

  • Toolbar

Custom toolbar

Summernote allows you to make own custom toolbar.

$('.summernote').summernote({
  toolbar: [
    //[groupname, [button list]]
     
    ['style', ['bold', 'italic', 'underline', 'clear']],
    ['font', ['strikethrough', 'superscript', 'subscript']],
    ['fontsize', ['fontsize']],
    ['color', ['color']],
    ['para', ['ul', 'ol', 'paragraph']],
    ['height', ['height']],
  ]
});

    ['insert', ['link', 'picture']]

});

Callbacks

Summernote support initialize callbacks and jquery's custom event style callbacks.

Callback only works with camel case string.

Lowercase string has been used for basic event name(ex: oninit, onenter, onfocus, onblur, onkeyup, onkeydown, onpaste). In contrast, callbacks name for advanced feature has been used with camel case string. This is inconsistent and confusing to use. So we rename all lowercase callback to camel case string.

You can use only camel case string for callback's name after v0.6.5.

onInit

$('#summernote').summernote({
  onInit: function() {
    console.log('Summernote is launched');
  }
});
summernote.init
$('#summernote').on('summernote.init', function() {
  console.log('Summernote is launched');
});

$('#summernote').summernote({

Plugin

Summernote supports simple plug-in system.

How to define

You can define a plugin with the $.summernote.addPlugin().
For detail, reference to API Docs 

// template, editor
var tmpl = $.summernote.renderer.getTemplate();
var editor = $.summernote.eventHandler.getEditor();

// add hello plugin 
$.summernote.addPlugin({
  // plugin's name
  name: 'hello',

  init : function(layoutInfo) { // run init method when summernote was initialized
    // layoutInfo.holder() is current summernote's jquery instance.
    var $note = layoutInfo.holder();
    
    // you can use jquery custom event. 
    $note.on('summernote.focus', function(customEvent, nativeEvent) { 
       showLayer(nativeEvent.target);   
    });
    $note.on('summernote.update', function(customEvent, nativeEvent) {
       var $boldButton = $(this).summernote('toolbar.get', 'bold');
       $boldButton.toggleClass('active').css({
         color : 'red'
       });
    });

    $note.on('summernote.blur', function(customEvent, nativeEvent) {
      hideLayer(nativeEvent.target);
      var $boldButton = $(this).summernote('toolbar.get', 'bold');
      $boldButton.removeClass('active').css({
        color : 'inherit'
      });
    });          
    $note.summernote('insertText', 'plugin start.');
  },
          
  buttons: { // define button that be added in the toolbar
    // "hello" button (key is a button's name)
    hello: function () {
          
      // create button template 
      return tmpl.iconButton('fa fa-header', {
        // set event's name to used as callback when this button is clicked
        // add data-event='hello' in button element
        event : 'hello',
        title: 'hello',
        hide: true
      });
    }
  },

  events: { // events
    // run callback when hello button is clicked
    hello: function (event, editor, layoutInfo, value) {
      // Get current editable node
      var $editable = layoutInfo.editable();

      // Call insertText with 'hello'
      editor.insertText($editable, 'hello ');
          
      // or 
      layoutInfo.holder().summernote("insertText", "Hello");
    }
  }
});

How to use

To use plugin, add button's name in the toolbar

<script>
$("#editor").summernote({
  toolbar : [
    ...
    ['group', ['hello']]   // add button name
    ...
  ]
});
</script>

Hint

You can make a own autocomplete in summernote. Please use the following attributes.

Property Return Description
load run when plugin is initailized
match RegExp check trigger string (ex - :(colon) is emoji trigger string)
content DOM or String make to insert content in summernote
template String make item template for hint popover
search create a collection for keyword
First, load hint plugin.
<script type="text/javascript" src="../plugin/summernote-ext-hint.js"></script>
Second, let's make a github emoji sample.

Hint plugin is able to define in summernote method.

 $('.summernote').summernote({
    plugin : {
        hint : {
            loadEmoji : function (data) {
                this.emoji = Object.keys(data);
                this.emojiLink = data;
            },

            // init plugin
            load : function($popover) {
                var self = this;
                $.getJSON('https://api.github.com/emojis', function (data) {
                   self.loadEmoji(data);
                });
            },

            // check trigger string   (  ':'(colon) for emoji )
            match : /\B:([\-+\w]*)$/,

            // make to insert content in summernote
            content : function(item) {
                var url = this.emojiLink[item];
                if (url) {
                // dom object or text
                 return $('<img />').attr('src', url).css({ width : 20 })[0];
                }
                return "";
            },

            // show item for hint popover
            template : function(item) {
                var content = this.emojiLink[item];
                return '<img src="' + content + '" contenteditable="false" width="20" /> :' + item + ':';
            },

            // create a collection for keyword
            search : function(keyword, callback) {
                callback($.grep(this.emoji, function (item) {
                  return (item.indexOf(keyword)  === 0);
                }));
            }
        }
    }
});
